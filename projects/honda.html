<-- Kembali

Aplikasi rekomendasi karyawan untuk pelatihan di PT Daya Adicipta Motora


Aplikasi berfungsi untuk menentukan karyawan pada pelatihan berdasarkan kelompok jabatan karyawan tersebut

- Admin

  • Dashboard
  • Kelola data Profile
  • Kelola data Menu
  • Kelola data Trainer
  • Kelola data karyawan
  • Import Karyawan dari file Excel
  • Kelola data jabatan
  • Export Karyawan ke Excel dan PDF

- Trainer

  • Kelola data Profile
  • Kelola data karyawan berdasarkan kelompok jabatan training
  • Kelola data batch
  • Export Karyawan ke Excel dan PDF

- Teknologi:

  • PHP
  • Codeigniter
  • jQuery
  • MySQL
  • DataTables
